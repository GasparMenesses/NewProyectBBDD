# Informe Individual - Gaspar Menesses

Este proyecto me sirvió muchísimo para entender de forma práctica cómo gestionar una base de datos y conectarla para que pueda ser utilizada desde una aplicación real. 

# Tarea principal
Si bien mi tarea principal dentro del grupo estuvo enfocada en el desarrollo del frontend y en armar la interfaz, como equipo nos organizamos de tal manera que todos pudimos meter mano, aprender y entender el funcionamiento general del proyecto completo.

# Dificultades
Una de las principales dificultades que tuve durante el desarrollo fue terminar de entender cómo funciona todo el flujo usando Docker. Al principio me costó agarrarle la mano a cómo se comporta la base de datos cuando está en un contenedor y cómo se relaciona eso con los scripts de SQL que teníamos que ejecutar para levantar y popular las tablas. Fue un desafío, pero una vez que logramos hacerlo funcionar, me quedó mucho más claro el concepto de contenedores y cómo facilitan el despliegue del proyecto.

# Conclusión
En resumen, me llevo un gran aprendizaje sobre la integración de tecnologías (frontend, backend y base de datos) y sobre el uso de herramientas como Docker para estandarizar entornos de trabajo.
